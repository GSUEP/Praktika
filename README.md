   **Практическое задание по модулю МДК 04.02 «Обеспечение качества функционирования компьютерных систем».**
      1. Разработать сценарии внедрения программного продукта для рабочего места в соответствии с ГОСТ Р ИСО/МЭК 1207. Оформить план внедрения.
      2. Разработать руководство оператора в соответствии со стандартом. Оформить руководство с учетом требований к технической документации.
      3. Разработать (подготовить) документацию и отчётные формы для внедрения программных средств в соответствии со стандартом.
      4. Измерение и анализ эксплуатационных характеристик качества программного обеспечения. Разработать (подготовить) документацию отражающую анализ эксплуатационных характеристик качества программного обеспечения. Выявление причин возникновения проблем
      5. Выявление и документирование проблем установки программного обеспечения. Выявление причин возникновения проблем оформление сопровождающей документации, рекомендации по исправлению возникающих проблем.
      6. Устранение проблем совместимости программного обеспечения, выявление причин проблем совместимости, рекомендации по исправлению проблем совместимости.
      7. Конфигурирование программных и аппаратных средств по заданному алгоритму.
      8. Настройка системы обновлений программного и аппаратного обеспечения с учетом требований заказчика.
      9. Создание образа системы. Восстановление системы, устранение проблем в работе системы. Создание рекомендаций по предотвращению сбоя системы.
      10. Разработка модулей программного средства в соответствии с запросом заказчика, с учетом требований. Подготовка технической документации на разработку программного модуля.
      11. Настройка сетевого доступа пользователей к системе, разграничение доступа и обеспечение безопасности системы.
      12. Тестирование программных продуктов, анализ работоспособности и жизненного цикла. Выявление ошибок. Исправление ошибок, отладка программного средства с учетом выявленных проблем.
      13. Сравнение результатов тестирования с требованиями технического задания и/или спецификацией Исправление ошибок, отладка программного средства с учетом выявленных проблем.
      14. Анализ рисков работоспособности программного продукта, рекомендации по устранению и предотвращению рисков.
      15. Выявление первичных и вторичных ошибок. Исправление ошибок, отладка программного продукта с учетом выявленных ошибок.
      16. Обнаружение вируса и устранение последствий его влияния. Созданий рекомендаций или алгоритма защиты системы от вирусов.
      17. Установка и настройка антивируса. Настройка обновлений с помощью зеркала. Созданий алгоритма по настройке антивирусной защиты, ее постоянного обновления.
      18. Настройка политики безопасности компьютерной системы с учетом специфики объекта.
      19. Настройка браузера, его эксплуатация, способы настройки с учетом использования на объекте.
      20. Работа с реестром, его настройка, исправление ошибок.
      21. Работа с программой восстановления файлов, резервное копирование, сохранение и восстановление информации, очистки дисков.
